Make your WordPress site WCAG 2.1 AA compliant with a 1-line accessibility widget. No coding required.

== Description ==

**Inkluyo** is the easiest way to make your WordPress site accessible to everyone. Install with one click, get instant WCAG 2.1 compliance, and let your visitors customize your site to their needs.

Unlike heavy accessibility overlays or complex plugins, Inkluyo is:
- ⚡ **Ultra-lightweight** (~7KB gzipped, loads asynchronously)
- 🎯 **Non-intrusive** - discreet button, doesn't change your design
- 🌍 **Multi-language** - EN, ES, PT built-in
- ♿ **WCAG 2.1 AA Certified** - legally compliant
- 🎨 **Fully customizable** - color, position, language per your brand
- 📊 **Analytics included** - see how your visitors use accessibility features

= What Accessibility Features Does It Include? =

* **Dyslexia Mode** - Switches to highly legible, dyslexia-friendly typefaces
* **High Contrast Mode** - Intelligently inverts colors for better readability
* **Text Size Adjustment** - 100% to 200% scaling
* **Line Height & Letter Spacing** - Fine-tune text readability
* **Calm Mode** - Pauses all animations and transitions
* **Big Cursor** - Enlarges cursor for better visibility
* **Focus Indicators** - Highlights keyboard navigation
* **Dark Mode** - Eye-friendly dark theme

== Installation ==

1. Go to **Plugins → Add New** in WordPress admin
2. Search for "Inkluyo Accessibility"
3. Click **Install Now** → **Activate**
4. Go to **Settings → Inkluyo** to customize
5. Choose widget color, position, and language

That's it! Your site is now more accessible.

== Frequently Asked Questions ==

= Will this slow down my site? =
No. Inkluyo loads asynchronously (doesn't block page rendering) and is only ~7KB gzipped. Your Lighthouse scores won't be affected.

= Is this WCAG 2.1 AA compliant? =
Yes. The widget helps your site meet WCAG 2.1 AA standards. However, for full compliance, your site content (text, images, forms) should also follow WCAG guidelines.

= Do you sell user data? =
Absolutely not. We don't track users or sell data. Your analytics are private.

= Can I remove the "Powered by Inkluyo" badge? =
Yes! You can hide the badge in the plugin settings. We just appreciate attribution.

= What browsers are supported? =
All modern browsers: Chrome, Firefox, Safari, Edge. Mobile browsers too (iOS, Android).

= Does it work with my theme? =
Yes. Inkluyo works with all WordPress themes (Bootstrap, Divi, Elementor, etc.) because it's theme-agnostic.

= Can I customize colors and position? =
Yes! In Settings → Inkluyo, choose your brand color and position (bottom-right, bottom-left, etc.).

= Is there a paid version? =
This free version includes all core features. We offer **Pro** (advanced analytics) and **Enterprise** (white-label, multi-domain) plans at [inkluyo.com](https://inkluyo.com), but the free version is fully functional.
